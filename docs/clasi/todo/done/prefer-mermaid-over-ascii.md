# Prefer Mermaid Over ASCII for Diagrams

For overview documents, architecture documents, and the sprint technical
plan, add guidance that Mermaid diagrams are preferred over ASCII art
when a diagram is a reasonable choice.

This applies to any document where system structure, data flow, or
component relationships are being illustrated. Mermaid renders natively
in GitHub and most markdown viewers, making it more readable and
maintainable than hand-drawn ASCII.

## Where to add guidance

- The `overview.md` template or instructions
- Architecture document templates/instructions
- The `sprint-technical-plan.md` template or instructions
- Possibly the `elicit-requirements` or `create-technical-plan` skill
  definitions, so agents are reminded at authoring time
